## 복합 맥락 처리

| 표현 | 맥락 조합 | 결과 키워드 |
|------|----------|-------------|
| 20대 여자친구랑 데이트 | 20대 여자 + 데이트 | 감성카페, 브런치, 분위기 좋은 |
| 직장인 점심 회식 | 직장인 + 회식 | 고기집, 역세권, 룸 |
| 대학생 MT | 대학생 + 모임 | 가성비, 단체, 바베큐 |
| 가족 아이랑 | 가족 + 아이 | 키즈, 패밀리, 넓은 |
