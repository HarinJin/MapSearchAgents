# 음식 관련 은어 사전

Translator Agent가 참조하는 음식 관련 은어/맥락 표현 사전입니다.

## 은어 → 검색 조건 매핑

### 속이 편한

| 항목 | 값 |
|------|-----|
| **의미** | 소화가 잘 되고 속이 편안해지는 음식 |
| **검색 키워드** | 해장국, 죽, 백반, 설렁탕, 곰탕, 한정식, 나물, 채식, 두부 |
| **카테고리** | FD6 (음식점) |
| **제외 키워드** | 칼국수, 우동, 라면, 파스타 (밀가루 면류는 속이 부대낄 수 있음) |
| **리뷰 체크** | 속편한, 담백, 건강, 나물 |
| **신뢰도** | 0.95 |

**연관 표현**: 부드러운, 담백한, 자극적이지 않은, 건강한

**참고**: 한정식은 나물류가 많아 소화가 잘 되므로 "속이 편한" 범주에 포함

## 복합 표현 처리

| 표현 | 분해 | 결합 키워드 |
|------|------|-------------|
| 저렴하고 속 편한 | 가성비 + 속이 편한 | 백반, 죽, 한정식 |
| 혼술하면서 노가리 | 혼술 + 노가리 | 이자카야, 1인 호프 |
| 해장하고 싶은데 가성비 | 해장 + 가성비 | 콩나물국밥, 해장국 |
| 건강하고 속 편한 | 건강식 + 속이 편한 | 한정식, 나물, 채식 |

## 부정 표현 처리

| 부정 표현 | 해석 | 제외 키워드 |
|----------|------|-------------|
| 기름진 거 말고 | 담백한 음식 | 튀김, 치킨, 삼겹살 |
| 매운 거 빼고 | 순한 음식 | 떡볶이, 불닭, 마라 |
| 비싼 데 말고 | 가성비 | 파인다이닝, 오마카세 |
| 밀가루 말고 | 면류 제외 | 칼국수, 우동, 라면, 파스타, 국수 |
| 속 부대끼는 거 말고 | 소화 잘 되는 음식 | 칼국수, 우동, 튀김, 기름진 음식 |
